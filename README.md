<!-- GETTING STARTED -->

### Instalace

1. Install packages
```sh
  yarn install 
  ```
2. In the project directory, you can run:
  ```sh
  yarn dev
  ```
3. Runs the app in the development mode Open [http://localhost:1234/](http://localhost:1234)
